# Air Conditioning English Learning App

This is a simple, production-ready English learning web app designed for A1–A2 learners focusing on vocabulary, grammar, reading comprehension, and speaking skills using a passage about air conditioning.

## Features
- **Vocabulary Section**: Teaches 10 specific vocabulary words/phrases with simple definitions, Vietnamese translations, example sentences, and mini quiz questions.
- **Grammar Section**: Includes exercises on Past Simple, Passive Voice, Cause-effect, and Comparatives based on the passage.
- **Reading Comprehension**: Tests understanding of the passage with multiple-choice questions.
- **Speaking Section**: Prompts users to speak about topics related to the passage and provides feedback.

## Tech Stack
- React.js for building the UI.
- Tailwind CSS for styling.
- Chakra UI for Tabs component.
